# expbox

酒店订房通知单辅助插件，自动生成确认信息和变更请求模板。

## 功能

- 支持 hk.et818.com 和 vip2.lyouoa.com
- 自动提取团号、导游、用房信息
- 生成标准化的通知文本模板
- 左右两侧悬浮显示，方便复制使用

## 输出示例

**确认信息**：
```
兰州长信国际酒店（4星）:
	3月7日, 用房: 3 标准间
麻烦收到确认回复,谢谢
```

**变更请求**：
```
兰州长信国际酒店（4星）:
团号: LJ-SDY-XB-DMHY-260307(安安报价）
导游: 李静 18709425313
原订:
现麻烦变更为:
	3月7日, 用房: 3 标准间
确认变更回复，谢谢
```

## 安装

1. 下载或克隆本项目
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目目录

## 使用

1. 打开支持的网站页面
2. 点击浏览器工具栏中的 expbox 图标
3. 点击"显示"按钮生成通知模板
4. 点击"隐藏"按钮移除模板
